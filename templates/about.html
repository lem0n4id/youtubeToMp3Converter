This was a project made by lem0n4id and pspiagicw for the Xhacks hackathon. This project is an attemp to build a web app dedicated to convert youtube videos into audio(mp3) files.

The XHacks Hackathon was a 36 hour competition to build a application to build a app.The hackathon was a very good experience for us. We definately faced a lot of challenges but the Hackathon officials were quick to guide and help us. Thanks to them for such a great experience.And thanks to the sponsors of the Hackathon for supporting such a endevour.

The project was built using Flask and basic HTML and CSS. It uses youtube-dl in the background to download and transfer the required files.Thanks to lem0n4id for the excellent work on the back-end and deployment.
